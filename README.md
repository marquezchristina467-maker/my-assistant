# Nexus AI - Your Multi-Persona Live Assistant

Nexus AI is a cutting-edge, real-time voice assistant powered by the Gemini 2.5 Flash model. It features a specialized "Document Vault" for generating, storing, and managing structured information like forms, lists, and documents.

## 🚀 Features

- **Real-Time Voice Interaction**: Seamless, low-latency voice conversations with multiple AI personas.
- **Specialized AI Personas**:
  - **General Assistant**: Your friendly everyday companion.
  - **Academic Tutor**: Expert guidance for college-level subjects.
  - **Web Architect**: Senior developer for building and debugging websites.
  - **Cyber Security**: Ethical hacking and digital privacy consultant.
  - **Life Manager**: Organized scheduling and task management.
  - **Travel Planner**: Detailed itineraries and travel logistics.
  - **Task Boy**: Specialized in generating structured lists, forms, and schedules.
- **Document Vault**: Automatically captures and stores generated text, allowing for easy copying and printing.
- **Modern Dark UI**: A sleek, responsive interface designed for both desktop and mobile use.
- **Privacy Focused**: Direct connection to Gemini API with secure session management.

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Gemini API Key (get one at [ai.google.dev](https://ai.google.dev))

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/nexus-ai-assistant.git
   cd nexus-ai-assistant
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your environment variables:
   Create a `.env` file in the root directory and add your API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## 📜 Usage

1. Select a persona from the sidebar that fits your needs.
2. Click **CONNECT NEXUS** to establish a neural link.
3. Speak clearly to the assistant. If you need a document, ask **Task Boy** to generate a list or form.
4. Use the **Document Vault** on the right to copy or print your generated content.
5. Click **TERMINATE LINK** when you are finished.
